# Economic Growth Forecasting

This project forecasts GDP growth for China, Germany, India, Japan, and the
United States using Meta Prophet.

## Project period

- Historical data: 1975–2025
- Model test period: 2021–2025
- Forecast period: 2026–2030

## Technologies

- Python
- Pandas
- Prophet
- Scikit-learn
- Streamlit
- Plotly
- Power BI
- Google Colab

## Model evaluation

The forecasting models were evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Percentage Error (MAPE)

## Dashboard pages

1. Economic Growth Overview
2. Country Forecast Analysis
3. Model Evaluation

## Run locally
